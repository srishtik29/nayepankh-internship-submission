## 2. NayePankh AI Assistant

A Gemini-powered chatbot built using Python, LangChain, and LangGraph.

### Features
- Gemini 2.5 Flash Integration
- Tool Calling
- Conversation Memory
- Volunteer Registration Workflow
- Foundation Knowledge Base

### Technologies
- Python
- LangChain
- LangGraph
- Gemini API

### Run

```bash
pip install -r requirements.txt
python chatbot.py
```
